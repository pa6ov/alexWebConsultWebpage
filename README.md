## 🚀 Overview: Your Digital Partner in Web Solutions

This repository contains a fully structured, multi-page prototype website designed for **AlexWeb Consult**, a premium tech consulting and full-cycle web development agency. 

The entire project is built from scratch using foundational technologies (HTML5, CSS3, Vanilla JavaScript) to showcase mastery over modern web standards, achieving peak performance, flawless responsiveness, and excellent Search Engine Optimization (SEO). 

This repository serves as an exemplary demonstration of our capabilities, ranging from simple brochure sites to complex, scalable e-commerce platforms.

> 💡 **Project Goal:** To create a polished, multi-page marketing suite that doesn't just *look* good, but *performs* flawlessly under technical and conversion metrics.

---

## ✨ Core Features & Technical Excellence

The design and architecture are intentionally advanced to prove mastery over best practices:

*   🎨 **Semantic Structure (HTML5):** Utilizes correct semantic tags (`<header>`, `<main>`, `<section>`, `<footer>`, etc.) for maximum accessibility and SEO value.
*   🔄 **Full-Cycle Interactivity:** Includes a dedicated `script.js` file handling advanced interactions like sticky navigation, smooth scrolling, and scroll-reveal effects (Intersection Observer API).
*   📱 **Perfect Responsiveness:** Built with mobile-first principles (`@media` queries in CSS) to ensure optimal viewing and usability on all devices (desktop, tablet, mobile).
*   📈 **Advanced SEO Implementation:** Every major page includes dedicated `<meta>` tags, Open Graph/Twitter Schema markup, JSON-LD Structured Data (e.g., `Organization`, `LocalBusiness`), and a comprehensive `sitemap.xml` to guarantee high search engine crawlability.
*   🧱 **Component Reusability:** The navigation bar, footer, and CTA sections are designed as shared components across all five files for consistency and rapid development.

## 📂 Project Structure (The Files)

This project is structured into highly modular, interconnected files:

| File Name | Purpose / Functionality | Key Output Showcase |
| :--- | :--- | :--- |
| **`index.html`** | **Homepage:** The entry point. Features a compelling hero section and service pillar overview to hook the visitor immediately. | High-conversion CTA block, initial brand exposure. |
| **`services.html`** | **Services Page:** Deep dive into core offerings (Website Creation, Tech Consulting, Maintenance). Uses two-column layouts to break down complex topics. | Detailed list management (`<ol>`, `<ul>`) and strategic service segregation. |
| **`about.html`** | **About Us:** Establishes authority. Features the company story and a detailed "Team Member" profile (Alex Pashov) to showcase senior expertise. | Credibility building, professional bio layout, and tech stack visualization. |
| **`portfolio.html`** | **Work Portfolio:** Showcases 3-4 key client projects using distinct cards with different metadata (industry tags, links). | Visual design mastery, structured content grid system, and live project linking. |
| **`contact.html`** | **Contact Page:** The conversion funnel endpoint. Features a fully structured form (`<form>`) for lead capture and direct contact information. | Form handling best practices, clear calls-to-action (CTA), and final submission points. |
| **`styles.css`** | **Styling Engine:** Central CSS file containing all custom properties, global resets, component styles, and responsive media queries. | Visual polish, maintainability, and brand consistency implementation. |
| **`script.js`** | **Behavior Layer:** Contains all JavaScript logic (Menu toggle, scroll effects, sticky nav handling). Uses vanilla JS for performance. | Interactivity, dynamic UX improvements, and state management simulation. |
| **`sitemap.xml`** | **SEO Tool:** Ensures search engines crawl the entire site structure efficiently. | SEO compliance. |
| **`robots.txt`** | **Crawl Management:** Directs search engine bots on which directories to index and how deeply to crawl. | SEO compliance. |
